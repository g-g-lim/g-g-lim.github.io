# 큰수만들기

<img src="/assets/images/큰수만들기.png" width="100%" alt="큰수만들기" />

```python   
def solution(number, k):
    stack = []
    for i in range(len(number)):
        while stack and k > 0 and stack[-1] < number[i]:
            stack.pop()
            k -= 1
        stack.append(number[i])

    if k > 0:
        stack = stack[:-k]
    return "".join(stack)
```
