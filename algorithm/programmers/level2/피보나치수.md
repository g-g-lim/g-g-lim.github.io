# 피보나치수

```javascript
function solution(n) {
    if (n === 0) {
        return 0;
    } else if (n === 1) {
        return 1;
    }
    let a = 0;
    let b = 1;
    for (let i = 0; i < n - 1; i++) {
        let temp = b;
        b = (a + b) % 1234567;
        a = temp;
    }
    return b;
}
```
