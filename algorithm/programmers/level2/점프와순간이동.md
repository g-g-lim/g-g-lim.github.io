# 점프와순간이동

<img src="/assets/images/programmers/점프와순간이동.png" width="100%" alt="점프와순간이동" />


```javascript
function solution(n){
    let answer = 1;
    while (n > 2) {
        if (n % 2 === 0) {
            n  = n / 2;
        } else {
            n -= 1;
            answer += 1;
        }
    }
    return answer;
}
```
