# 멀리뛰기

<img src="/assets/images/멀리뛰기.png" width="100%" alt="멀리뛰기" />

```javascript
function solution(n) {
    const answer = [1,2];
    for (let i = 2; i < n; i++) {
        let a = answer[answer.length - 1] + answer[answer.length - 2]
        answer.push(a % 1234567);
    }
    return answer[n - 1];
}
```
